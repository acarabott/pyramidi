# Pyramidi MIDI to OSC converter: PM_MIDI2OSC

## Overview

### PM_MIDI2OSCChannel 

- Represents a single MIDI to OSC converter.
- The following are configurable
    + MIDI Device / Port
    + MIDI Channel
    + MIDI Message Type
    + IP / Port to send to
    + OSC address to use
    + Latency to send with (for synchronisation)
